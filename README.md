# HRRR_Jupyter

This is a project to allow anyone to produce charts from NCEP's 2.5km HRRR model using a Jupyter Notebook.

## Install 
run `./setup_hrrr.sh` 

or `conda env create -f environment.yml`

## Use
run `./run_hrrr.sh` 

or `conda activate hrrr` then `jupyter lab`
